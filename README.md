# DreamCation-Repo
DreamCation is a travel itinerary web application where you can search for your dream vacation, save it in a plan and share these plans with your friends! Utilizes React for Frontend, Express and MySQL for backend.


To run this project locally:

Clone the repository:
git clone https://github.com/AspiredProgrammer/DreamCation-Repo.git
cd <your-project-folder>

Run the install script:
npm install

Start the development server:
npm run dev
